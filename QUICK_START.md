# Quick Start - School Clinic Patient System

## What Changed?

Your clinic's patient management system has been completely redesigned to support students, teaching staff, and non-teaching staff with specific information for each group.

## Key Features at a Glance

✨ **Three Patient Types**
- Students (with program, course, year, block)
- Teaching Staff (with program affiliation)
- Non-Teaching Staff (with department category)

✨ **Smart Sorting**
- Sort by name, role, or ID number
- Ascending/descending options

✨ **Better Data**
- Uses date of birth instead of age
- Separated name fields for better organization
- Structured ID numbers per role

✨ **Validation**
- Names: Letters only ✓
- Phone: 09 + 11 digits ✓
- ID: Letters, numbers, dashes ✓

## Getting Started

### 1. Add Your First Patient

1. Click **"Add Patient"** button
2. Fill in **Personal Information**:
   - Name (first, middle, last, suffix)
   - Birth date
   - Gender & contact info
3. Select **Role** (Student/Staff/Non-Staff)
4. Fill in **School Information**:
   - ID number
   - Role-specific details
5. Click **"Add Patient"**

### 2. View All Patients

- Patients appear in a sorted list below the form
- Each card shows all relevant info
- Click **Edit** or **Delete** to manage

### 3. Find Someone

- Use the **Search** box to find by name or ID
- Select **Sort By** to organize the list
- 6 sorting options available

## Form Tips

### Student Form Includes:
- Program: Pick CICT or CBME
- Course: Choices change based on program
- Year: 1st through 4th year
- Block: 1 through 5

### Teaching Staff Form Includes:
- Program: CICT or CBME

### Non-Teaching Staff Form Includes:
- Category: Admin, Security, Maintenance, Support, or Clinic

## Important Notes

### Names
- Only letters allowed (no numbers like "Smith123")
- Use spaces between first/middle names
- Middle name & suffix are optional

### Phone Numbers
- Must start with **09**
- Must be exactly **11 digits**
- Must be all numbers
- Example: **09123456789**

### ID Numbers
- Can have letters, numbers, and dashes
- Examples: **STU-2024-001**, **TS-CICT-01**
- Each role can use its own numbering system

### Date of Birth
- Use the calendar picker
- Easier than entering age (stays current automatically)

## Common Tasks

### Add a Student
1. Click "Add Patient"
2. Enter name and personal info
3. Select "Student" as role
4. Enter student ID
5. Pick program (CICT or CBME)
6. Pick course (updates based on program)
7. Pick year level (1-4)
8. Pick block (1-5)
9. Click "Add Patient"

### Find All Students in Block 3
1. Click "Sort By" → "Role (A-Z)"
2. Students appear first
3. Look for entries with Block 3

### Find a Teacher
1. Click "Sort By" → "Role (A-Z)"
2. Teaching Staff appears in middle
3. Or use search with teacher's name

### Update Patient Info
1. Find patient in list
2. Click "Edit"
3. Change any field
4. Click "Update Patient"

### Remove a Patient
1. Find patient in list
2. Click "Delete"
3. Confirm when asked
4. Patient is removed

## Sorting Options Explained

| Option | What It Does |
|--------|--------------|
| **Name (A-Z)** | Lists patients alphabetically by name |
| **Name (Z-A)** | Lists patients reverse alphabetically |
| **Role (A-Z)** | Groups by role: Students, Staff, Non-Staff |
| **Role (Z-A)** | Groups by role in reverse order |
| **ID (Ascending)** | ID numbers from lowest to highest |
| **ID (Descending)** | ID numbers from highest to lowest |

## Error Messages Guide

**"Name can only contain letters"**
- Remove numbers/special characters from name

**"Phone must start with 09 and be exactly 11 digits"**
- Format: 09 + 9 more digits (09123456789)

**"ID number can only contain letters, numbers, and dashes"**
- Don't use spaces or special characters

**"Program is required for students"**
- Select CICT or CBME for student records

## Troubleshooting

**Patient not saving?**
- Check for error messages in red
- Ensure all required fields are filled
- Phone must be 11 digits starting with 09

**Can't find a patient?**
- Use the search box
- Try different sort options
- Check spelling of name or ID

**Form looks different for some patients?**
- This is normal! Fields change based on role
- Students see different options than staff
